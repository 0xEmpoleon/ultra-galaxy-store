## Fonts

### Twobit (Exceptional Display) — OTF

| File | Weight | Use |
|---|---|---|
| `fonts/twobit/Twobit Bold.otf` | Bold | Dramatic brand expression. Use sparingly. |
| `fonts/twobit/Twobit.otf` | Regular | Dramatic emphasis |
| `fonts/twobit/Twobit Italic.otf` | Italic | Dynamic effect |
| `fonts/twobit/Twobit Bold Italic.otf` | Bold Italic | High-impact emphasis |

### Source Code Pro (Body/UI) — Variable TTF

| File | Notes |
|---|---|
| `fonts/Source_Code_Pro/SourceCodePro-VariableFont_wght.ttf` | ✅ **PRIMARY** — variable font, full weight range 200–900 |
| `fonts/Source_Code_Pro/static/` | Individual weight TTFs for environments that don't support variable fonts |

> **CDN alternative:** `https://fonts.googleapis.com/css2?family=Source+Code+Pro:ital,wght@0,200..900;1,200..900`

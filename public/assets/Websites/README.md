## Starting a New Web Project

1. Create a new subfolder: `Websites/[project-name]/` (use kebab-case)
2. Follow the workflow in `.agent/workflows/build-website.md`
3. Add the project to the table above once it's underway
4. Include a `README.md` inside the project folder with a brief description and status

## Design Baseline for Every JOY Web Project

- **Background:** Soft Dark `#242424` or Base Light `#E0E0E0`
- **Accent / CTA:** Signal Lime `#EDFFAD`
- **Text / UI:** Circuit Grey `#E0E0E0` (on dark mode)
- **Display font:** Source Code Pro (SemiBold)
- **Body font:** Source Code Pro (Google Fonts CDN)
- **Default render:** `Brand Guidelines/renders_2d/Right Front JOY Genesis.png`
- **Default logo:** `Brand Guidelines/logos/Joy_Logo_Text_1.png`
